Gestion de Services IT

Ce projet porte sur l’installation et la gestion de services informatiques pour une entreprise.

Services et technologies abordés :

  • Installation et configuration de **GLPI** pour la gestion des tickets IT.
  • Mise en place d’un serveur **NextCloud** pour le stockage et le partage sécurisé.
  • Déploiement et configuration d’un **serveur WordPress** sous Debian 11.
  • Gestion des ressources de stockage avec **FSRM** sur Windows Server.
  • Installation et configuration du pare-feu **IPFire**.

Documents du projet :

  • Installation et configuration de GLPI
  • Installation de Nextcloud sur Debian
  • Gestion des quotas avec FSRM
  • Déploiement d’un serveur WordPress
  • Installation et configuration d’IPFire

Gestion des Stratégies de Groupe (GPO)

Ce projet a été réalisé pour centraliser la gestion informatique d'une agence de voyage.

Étapes du projet :

  • Création et configuration d'un Contrôleur de Domaine sous Windows Server 2019.
  • Ajout et configuration des machines clientes sous Windows 10.
  • Organisation des utilisateurs en Unités Organisationnelles (OUs).
  • Définition et application des stratégies de groupe (GPO) pour chaque service.
  • Gestion des permissions NTFS et accès aux ressources partagées.
  • Configuration du pare-feu avec IPFire/PfSense.
  • Tests et validation de l'infrastructure.

Documents du projet :

  • Étape 1 : Mise en place d’Active Directory
  • Étape 2 : Configuration des OUs et GPO
  • Étape 3 : Gestion des permissions et ressources
  • Étape 4 : Sécurisation et tests
